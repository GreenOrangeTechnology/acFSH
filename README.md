# acFSH 小智智能家居Android
## 使用说明
1.开发环境 AndroidStudio 3.0  , JDK 1.8.0
2.基于百度语音识别bd_speech_sdk_asr_v3.0.3.20170801_60da871  demo
3.下载项目后导入as,真机调试时数据线连接手机

## 版本规划

       2.大致UI轮廓(Splash跳转 绿色环保风格 主界面直接显示六大功能模块 开始录音按钮模仿小米语音助手)
